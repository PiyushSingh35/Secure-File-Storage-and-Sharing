Create .env file in project root:

Step 5: Open PowerShell in Project Folder

cd "D:\CV PROJECT\Secure File Storage"

Step 6: Initialize Git

git init
git add .
git commit -m "Initial commit: Secure File Storage & Sharing Application"

Step 7: Create Repository on GitHub

1. Go to https://github.com/new
2. Repository name: Secure-File-Storage
3. Description: End-to-end encrypted file storage and sharing application with AES-256-GCM
4. Select: Public
5. Click: Create repository

Step 8: Connect to GitHub

Replace YOUR_USERNAME with your actual GitHub username:

git remote add origin https://github.com/YOUR_USERNAME/Secure-File-Storage.git
git branch -M main
git push -u origin main

Step 9: Verify on GitHub

1. Go to https://github.com/YOUR_USERNAME/Secure-File-Storage
